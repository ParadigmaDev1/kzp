Ваше устройство не поддерживает автоматическую установку. Чтобы добавить сайт на главный экран, воспользуйтесь инструкцией снизу

1. Нажмите кнопку “Поделиться” в правом верхнем углу браузера

2. Во всплывающем меню коснитесь строки “Добавить в Dock”

3. При желании измените название веб-приложения или оставьте то, которое будет предложено по умолчанию

4. Нажмите кнопку “Добавить”.
